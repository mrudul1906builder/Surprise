Files
-----
index.html      Entry point
style.css       Styles (glassmorphism + animations)
script.js       Cinematic timeline, scenes, canvas animations
assets/music.mp3  Your uploaded song (used as-is)

Deploy to Netlify
-----------------
Option 1 (drag & drop):
  1. Go to https://app.netlify.com/drop
  2. Drag this entire folder onto the drop zone.
  3. Done. Share the URL.

Option 2 (Netlify CLI):
  netlify deploy --dir . --prod

No build step. No dependencies. Pure HTML / CSS / vanilla JS.

Notes
-----
- Tap the rose to begin (browsers require user interaction before audio can play).
- The proposal fades in ~22 seconds into the song. Use the Skip button in the
  top-left to jump straight to it.
- Mute/unmute is in the top-right.
- Best viewed on a phone. Works on desktop too.
